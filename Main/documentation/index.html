Install Node.Js

Your computer must have installed nodejs to run this template. We built this project using node version 20, so to avoid conflicting packages included in this project, we recommend using node version 20. You can download Node.js from here. If you go to the node homepage and don't see version 20, go to Download, then change to "Pre-built installer" and select version 20 and click Download to download nodejs. After downloading, you need to run the application to install Nodejs for your computer. NPM comes bundled with Node.js

Install Npm Packages

You must have to install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.

Convert Tailwindcss

After installed packages, run "npm run tailwindcss" from the root of your project to convert Tailwindcss to CSS. Now when you make changes to HTML, the browser automatically converts the Tailwindcss classes into CSS in the directory /dist/output-tailwind.css.

Convert SASS

Choose new Terminal, run "npm run sass" to convert SASS to CSS. Now when you want to change style to elements, let change .scss files in /assets/scss/. the browser automatically converts the SCSS file into CSS in the directory /dist/output-scss.css.

Run project with live server

To run the project like on the actual server, you need to install the "Live Server" Extensions(Ctrl + Shift + X) of VS Code. After installing, you can right-click on the .html file, then select Open with Live Server, the project will be running on the browser. Every time you change the project code, the browser will automatically change without you need to refresh
